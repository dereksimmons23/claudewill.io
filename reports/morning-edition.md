## AI & Technology

## Google Introduces TurboQuant, a Memory Compression Algorithm for Large AI Models

Google's research team unveiled **TurboQuant** at ICLR 2026 on April 2, an algorithm that significantly reduces memory overhead from the KV cache, a major bottleneck in running large language models.[1] The technique combines PolarQuant vector rotation with the Quantized Johnson-Lindenstrauss compression method to allow models with massive context windows to run more efficiently.[1] The breakthrough could accelerate a shift from raw parameter scaling to efficiency-first AI development, with implications for on-device AI and reducing data center costs.[1]

## Gemini 3.1 Ultra Ships with 2-Million Token Context and Native Multimodal Reasoning

Google launched **Gemini 3.1 Ultra** on March 20, featuring a 2-million token context window that works natively across text, image, audio, and video without transcription intermediaries.[1] Unlike prior Gemini versions, 3.1 was designed from training to reason across all modalities simultaneously and includes a sandboxed Code Execution tool allowing the model to write, run, and test code mid-conversation.[1] The model also features significantly improved grounding to reduce hallucinations on factual queries.[1]

## Google DeepMind's AlphaEvolve Advances Theoretical Computer Science

**AlphaEvolve**, a Gemini-powered coding agent that pairs large language models with evolutionary algorithms, has discovered new mathematical structures that improve state-of-the-art results on long-standing open problems in complexity theory.[1] The system has been deployed inside Google's infrastructure for over a year, recovering 0.7% of Google's worldwide computing resources continuously and speeding up a key kernel in Gemini's architecture by 23%.[1]

## Iran Publishes Satellite Coordinates of OpenAI's $30 Billion Data Center

Iran disclosed satellite coordinates of **OpenAI's $30 billion Stargate AI data center** in Abu Dhabi and threatened "complete annihilation" of the facility.[2] The threat represents a significant geopolitical escalation targeting critical AI infrastructure and follows recent disclosure of npm supply chain attacks attributed to North Korean actors.[2]

## Anthropic Cuts Off OpenClaw From Claude Subscriptions, Forcing API Pricing

**Anthropic** has closed developer access to Claude subscriptions through OpenClaw, forcing users to adopt API pricing instead.[2] The move sparked developer backlash but reflects Anthropic's strategy to standardize monetization across its platform.[2]
