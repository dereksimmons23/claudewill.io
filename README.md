# Claude Wisdom | claudewill.io

A digital repository bridging human wisdom with AI capabilities through frameworks, visual explorations, and practical applications.

## Overview

The Claude Wisdom platform serves as both showcase and laboratory for The CW Standard - a framework integrating pattern recognition methodologies across domains. This repository contains the code, content, and documentation for:

- **The CW Standard** - Framework for natural development and authentic transformation
- **LIGHTNING/BUG** - AI-generated visual exploration of technological evolution
- **FloState** - Performance optimization system for flow state achievement

## Project Structure

```
claude.io/

│   ├── components/         # Reusable UI components

│   └── media/              # Media assets and resources
```

## Local Development

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/claude.io.git
cd claude.io

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Deployment

This repository is configured to deploy to [claudewill.io](https://claudewill.io) via GitHub Actions. The deployment workflow is triggered on pushes to the main branch.

### Manual Deployment

```bash
# Build the production version
npm run build

# Deploy to claudewill.io
npm run deploy
```

## Content Management

Content for the framework documentation, project descriptions, and blog posts is managed through Markdown files located in the `content/` directory. Updates to these files will be reflected on the site after deployment.
